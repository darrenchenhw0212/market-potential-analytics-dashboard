# Data

## Raw Data

This project integrates one business dataset with three external economic indicators.

| Dataset | Description |
|---------|-------------|
| superstore.xlsx | Global Superstore sales and profit data |
| gdp_per_capita.csv | GDP per capita by country |
| population_by_country.csv | Population by country |
| internet_user_by_country.csv | Internet usage percentage by country |

## Processed Data

Processed datasets are generated by the preprocessing scripts and stored in `data/processed/`.
